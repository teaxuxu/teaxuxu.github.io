Unsere Aufgabe als Checkliste zur Orientierung:

  • (check) Mindestens 3 Seiten erstellen inkl. Landing Page (index.html) die miteinander verlinkt sind
  • (check) CSS Datei erstellen, mit rudimentärer visueller Gestaltung
  • (check) Webpages und Style Sheet müssen mit relativen Links verlinkt sein
  • (check) Mindestens 2 unterschiedliche Überschriften
  • (check) Mindestens einen Paragraphen mit Text
  • (check) Mindestens ein Bild - Breite passt sich Browserfenster an
  • (check) Bild als Link zu einer internen oder externen Seite (Bild mit Code ist verlinkt)
  • (check) Externer Link der im neuen Tab/Fenster geöffnet wird
  • (check) verschachtelte Liste mit zwei Ebenen oder eine Tabelle
  • (check) Ein iFrame/Videodatei/Audiodatei
  • (check) div oder span
  • (check) Zwei semantische Ebenen
  • (check) Attribute id
  • (check) Attribut class
  • (check) CSS: color/backround-color verwenden
  • (check) Überschriften h1 bis h6 fett und auf auf Verdana umstellen, Backup soll jede Sanserife-Schrift sein
  • (check) Schriftfamilie für normalen Text auf Georgia, mit beliebiger verfügbarer Serifenschrift als Backup
  • (check) Überprüfung mit validate
  • (check) Überprüfung mit jigsaw
  • (check) Am Ende pushen auf Git Hub und Moodle hochladen
